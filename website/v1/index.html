AIPlay Automated Debugger

Demonstrated here is our core game-play and debugging system, AIPlay (built in ai/v1/aiplay). AIPlay utilizes an Electron-based runtime environment coupled with state-of-the-art agentic LLMs to playtest, simulate actions, catch bugs, and automatically patch source files in real-time.

Contribute a Game

4weird is open source - ship your own weird game and get it featured here

1
🍴

Fork & Clone

Fork the repo on GitHub, then clone it locally. No build tools, no frameworks required - just open and play.

2
🎨

Build Your Game

Create a folder under website/v1/games/html/yourgame/ with an index.html, your JS, and CSS. Pure HTML5 - go weird.

3
🔗

Add a Game Card

Drop a new card into the Games section of the main page so players can launch your creation in one click.

4
🚀

Open a Pull Request

Push your branch and open a PR. We review fast, give feedback, and ship the weird ones live.

Got a weird idea? We want it. Every contributor gets credited.
