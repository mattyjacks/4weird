📖 How to Use

Step 1: Load Your Project

Click "Add Project" or paste your project folder path into the Quick Launch panel.

C:\Users\YourName\Projects\my-nextjs-app

aiplay will automatically detect your framework and project type.

Step 2: Start the Server

Click the "Start" button on your project card. aiplay will:

  • Launch the dev server
  • Find an available port
  • Display server status
  • Show live output

Step 3: Open & Debug

Click "Open" to view your app in the built-in webview, or access DevTools for debugging.

http://localhost:3000

Use the console panel to monitor server output in real-time.

Step 4: Manage & Reload

Use the server status panel to reload, view logs, or stop the server.

  • 🔄 Reload - Hot reload your app
  • 📋 Logs - View server output
  • 🔧 DevTools - Open Chrome DevTools
  • ⏹ Stop - Stop the dev server
